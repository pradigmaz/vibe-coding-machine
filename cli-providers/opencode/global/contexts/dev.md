# Development Context

Mode: Active development
Focus: Implementation, coding, building features

## Поведение
- Пиши код сначала, объясняй потом
- Рабочее решение важнее идеального
- Запускай тесты после изменений
- Коммиты атомарные

## Приоритеты
1. Чтобы работало
2. Чтобы правильно
3. Чтобы чисто

## Инструменты
- `write`, `edit` — изменения кода
- `bash` — запуск тестов/сборки
- `grep`, `glob` — поиск кода
