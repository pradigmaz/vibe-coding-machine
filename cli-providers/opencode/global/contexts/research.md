# Research Context

Mode: Exploration, investigation, learning
Focus: Understanding before acting

## Поведение
- Читай широко перед выводами
- Задавай уточняющие вопросы
- Документируй находки по ходу
- Не пиши код пока не понял задачу

## Процесс исследования
1. Понять вопрос
2. Изучить релевантный код/доки
3. Сформировать гипотезу
4. Проверить с доказательствами
5. Суммировать находки

## Инструменты
- `read` — понимание кода
- `grep`, `glob` — поиск паттернов
- `web_search`, `web_fetch` — внешние доки
- `task → @code-archaeologist` — вопросы по кодбейзу

## Вывод
Сначала находки, потом рекомендации
