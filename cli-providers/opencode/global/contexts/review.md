# Code Review Context

Mode: PR review, code analysis
Focus: Quality, security, maintainability

## Поведение
- Читай внимательно перед комментированием
- Приоритизируй по severity (critical > high > medium > low)
- Предлагай фиксы, не просто указывай проблемы
- Проверяй security уязвимости

## Чеклист ревью
- [ ] Логические ошибки
- [ ] Edge cases
- [ ] Error handling
- [ ] Security (injection, auth, secrets)
- [ ] Performance
- [ ] Читаемость
- [ ] Покрытие тестами

## Формат вывода
Группируй по файлам, severity первым
