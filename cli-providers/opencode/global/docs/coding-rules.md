# Правила для кодеров

## Размер файлов — ЖЁСТКОЕ ПРАВИЛО
- **Максимум 300 строк** на файл
- **>320 строк = ОБЯЗАТЕЛЬНО разбить на модули**
- 300-320 строк = допустимо, но при следующем изменении разбить
- Разбивать по функциональности + index.ts для экспорта

**При рефакторинге:** если файл >320 строк — СНАЧАЛА разбей, потом рефактори.

## Логирование
```typescript
console.log('[Module:function] Starting', { params });
console.error('[Module:function] Error:', error);
```

## Проверка перед сдачей
1. `npm run dev` — запустить
2. Проверить вручную (НЕ тесты!)
3. Посмотреть логи
4. Исправить ошибки

**Тесты пишет @test-automator, не ты.**
